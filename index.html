YouTube Downloader

About

A simple, responsive YouTube video downloading client

Made by Gerald Wu, powered by youtube-dl.

Can I download videos other than YouTube videos?
Actually, yes! I just used YouTube in the name because it's the most recognized platform. However, any web video platform supported by youtube-dl will work just fine. The full list is here.

How does this work?
Simply enter the YouTube URL you want to download and hit the 'download' button. After a few moments, a download link will appear with your video.

No, really, how does this work?
When you hit download, the page sends an AJAX request to a server-side .php file, along with the URL you entered. This URL is passed on to a shell script that runs server side to cache the video using the youtube-dl program.
After it's cached, the URL is passed back to client for you to download. The file remains cached until midnight, when a cronjob clears the cache folder.
If you want to delve into the code yourself, the source code is available on GitHub here. Or, you can close this box and click the GitHub triangle in the upper right-hand corner.

Why did you make this?
I wanted an internet-connected, self-hosted YouTube downloading service. While I could use vanilla youtube-dl, I won't always have access to a machine with it installed. And other online YouTube downloading sites are sketchy and ad-ridden, to put it mildly. I can't really trust them. Thus, this project was born.
Also, it was a nice exercise to get myself more familiar with fundamental web development.
